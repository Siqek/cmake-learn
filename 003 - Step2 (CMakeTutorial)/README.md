# Learned Commands

| command                  | explanation |
| ---                      | --- |
| set()                    | sets the value of a variable |
| message()                | prints a message |
| list()                   | performs operations on a list |
| if()                     | conditionally executes a group of commands |
| foreach()                | executes a group of commands for each item in a list |
| macro()                  | defines a reusable block of commands that executes in the caller's (parent) scope |
| function()               | defines a reusable block of commands that executes in its own local scope |
| include()                | loads and executes CMake code from the specified file |
